Manage your subscription, billing, credits, and usage tracking.

## Subscription Plans

**Free**: Limited credits, core features, basic support
**Individual** (Starter/Pro/Premium): Monthly credits, priority support, extended storage
**Team**: Shared credits pool, team management, collaboration features
**Enterprise**: Custom pricing, dedicated support, SLA guarantees

## Managing Your Subscription

**Upgrade/Downgrade:**
1. Go to Subscription
2. Select new plan
3. Confirm - upgrades are immediate, downgrades take effect next billing cycle

**Cancel:**
1. Subscription → "Cancel Subscription"
2. Confirm - access continues until period ends
3. Reactivate anytime

**Payment Methods:**
- Add/update in Billing section
- Supports credit/debit cards, PayPal
- Enterprise: bank transfer, invoice billing

## Credits System

**How Credits Work:**
- Credits = generation currency
- Monthly allocation based on plan
- Different actions cost different amounts

**Typical Costs:**
- Image generation: 1-5 credits
- Video generation: 10-50 credits
- Audio generation: 2-10 credits
- Model training: 50-100 credits
- Varies by model and quality

**Add-On Credits:**
- Purchase one-time credit packs
- Never expire
- Volume discounts available

**View Usage:**
- Dashboard shows current balance
- Track usage by tool/project
- Set low-credit alerts

## Billing & Invoices

**Billing Cycle:**
- Monthly or annual (20% discount on annual)
- Auto-renewal on billing date
- View next charge date in dashboard

**Invoices:**
- Download PDF from Billing History
- Includes all charges, taxes, fees
- Email copies available

**Promo Codes:**
1. Go to Billing → "Apply Promo Code"
2. Enter code (e.g., KOLBO30 for 30% off)
3. Apply to subscription

## Storage Management

**Storage Quota:**
- Included in plan
- Used by generated media, uploads, projects
- View usage in dashboard

**Optimize Storage:**
- Delete unused media
- Archive old projects
- Export and remove when needed

## Team & Enterprise

**Team Billing:**
- Shared credits pool
- Per-user usage tracking
- Add/remove seats with pro-rated billing

**Enterprise:**
- Custom contracts and pricing
- Net 30/60 payment terms
- Detailed usage reports
- Department cost allocation

## Best Practices

1. Monitor credit usage regularly
2. Set up low-credit alerts
3. Use annual billing for 20% savings
4. Apply promo codes (KOLBO30)
5. Clean storage regularly
6. Choose plan based on actual usage
7. Track usage trends monthly
