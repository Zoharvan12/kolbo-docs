## Common Tasks

**Change Email:**
1. Navigate to Profile Settings
2. Click "Change Email"
3. Enter new email
4. Verify via email link
5. Confirm change

**Enable 2FA:**
1. Go to Security Settings
2. Click "Enable 2FA"
3. Scan QR with authenticator
4. Save backup codes
5. Verify setup

**Export Data:**
1. Go to Privacy Settings
2. Click "Export Data"
3. Choose data types
4. Confirm export
5. Download when ready

**Customize Interface:**
1. Open Interface Preferences
2. Select language/theme
3. Adjust display settings
4. Set tool defaults
5. Save changes
