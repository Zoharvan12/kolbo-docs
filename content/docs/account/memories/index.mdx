# User Memories

User Memories is Kolbo.AI's personal AI memory system that helps the AI understand you better and provide more personalized, context-aware assistance.

## Overview

Memories allow Kolbo's AI to remember your preferences, work style, project context, and common tasks - creating a truly personalized AI experience that improves over time.

## What Are User Memories?

### Personal AI Context
Information AI remembers:
- Your preferences and work style
- Project types and goals
- Common workflows
- Creative style and aesthetic
- Technical preferences
- Communication style

### How It Works
AI learns from:
- Explicit memories you add
- Patterns in your work
- Feedback and corrections
- Tool usage habits
- Project context
- Interaction history

### Benefits
Personalized experience:
- Faster, more relevant suggestions
- Context-aware responses
- Reduced repetition
- Improved accuracy
- Time-saving automation
- Consistent results

## Managing Memories

### Add Memory
Create new memories:
1. Navigate to User Memories
2. Click "Add Memory"
3. Enter memory text
4. Categorize (optional)
5. Set scope (global/project)
6. Save memory

### Edit Memory
Update existing:
- Browse memories
- Select to edit
- Modify content
- Update category
- Save changes

### Delete Memory
Remove memories:
- Select memory
- Click delete
- Confirm deletion
- Memory removed
- AI forgets context

### Search Memories
Find specific memories:
- Search by keyword
- Filter by category
- Filter by project
- Sort by date
- Quick access
