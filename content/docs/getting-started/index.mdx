# Getting Started with Kolbo.AI

Welcome! This guide will help you get up and running with Kolbo.AI in minutes.

## Prerequisites

- A Kolbo.AI account (sign up at [app.kolbo.ai](https://app.kolbo.ai))
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- Basic understanding of AI concepts (helpful but not required)

## Account Setup

1. **Sign Up**: Visit [app.kolbo.ai](https://app.kolbo.ai) and create your account
2. **Choose a Plan**: Start with our free tier or upgrade for more credits
3. **Complete Onboarding**: Follow the quick tutorial to familiarize yourself with the interface

## Your First Project

The Kolbo.AI dashboard is organized around **projects**. Each project can contain:

- Images, videos, and audio files
- Chat conversations
- Custom trained models
- Workflow automations

### Creating a Project

1. Click "New Project" in the sidebar
2. Give your project a name and description
3. Start creating!
