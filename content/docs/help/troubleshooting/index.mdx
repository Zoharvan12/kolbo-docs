# Troubleshooting

Find solutions to common issues and learn how to resolve problems quickly.

## Quick Diagnostics

### Before Troubleshooting
Check these first:
1. **Internet Connection**: Stable connection required
2. **Browser**: Use latest Chrome, Firefox, Safari, or Edge
3. **Status Page**: Check service status at status.kolbo.ai
4. **Clear Cache**: Try clearing browser cache
5. **Different Browser**: Test in another browser

## Common Issues

### Account & Login Issues

#### Can't Log In
**Problem**: Unable to access account

**Solutions:**
- Verify email address
- Reset password
- Clear browser cache
- Try incognito mode
- Check email for verification link
- Contact support if locked out

#### Email Not Verified
**Problem**: Account requires email verification

**Solutions:**
- Check spam/junk folder
- Resend verification email
- Wait a few minutes
- Check correct email address
- Contact support if not received

#### Forgot Password
**Problem**: Can't remember password

**Solutions:**
1. Click "Forgot Password" on login
2. Enter email address
3. Check email for reset link
4. Create new password
5. Log in with new password

#### Two-Factor Authentication Issues
**Problem**: 2FA codes not working

**Solutions:**
- Verify correct code
- Check time sync on device
- Use backup codes
- Regenerate 2FA if needed
- Contact support to disable

### Generation Issues

#### Generation Failed
**Problem**: Content generation fails

**Solutions:**
- Check credit balance
- Verify prompt validity
- Try simpler prompt
- Check file size limits
- Try different model
- Clear cache and retry
- Contact support if persists

#### Slow Generation
**Problem**: Generation taking too long

**Solutions:**
- Check system status
- High demand periods = slower
- Try different time
- Reduce complexity
- Check connection
- Use faster models

#### Poor Quality Results
**Problem**: Results not meeting expectations

**Solutions:**
- Improve prompt detail
- Adjust quality settings
- Try different model
- Use reference images
- Check examples
- Increase resolution
- Follow best practices

#### Generation Interrupted
**Problem**: Generation stops mid-process

**Solutions:**
- Don't close tab during generation
- Stable internet required
- Check credit balance
- Check status page
- Retry generation
- Contact support if recurring

### Upload Issues

#### Can't Upload Files
**Problem**: File upload fails

**Solutions:**
- Check file size limit (varies by type)
- Verify file format supported
- Stable internet required
- Try smaller file
- Try different format
- Clear browser cache
- Try different browser

#### Upload Stuck
**Problem**: Upload progress bar stuck

**Solutions:**
- Wait a few minutes (large files)
- Cancel and retry
- Check internet connection
- Reduce file size
- Try different browser
- Contact support

#### Unsupported Format
**Problem**: Format not accepted

**Supported Formats:**
- Images: JPG, PNG, WebP, GIF
- Videos: MP4, MOV, WebM
- Audio: MP3, WAV, M4A
- Documents: PDF, DOCX, TXT

**Solutions:**
- Convert to supported format
- Check file isn't corrupted
- Verify file extension

## Browser-Specific Issues

### Chrome
- Clear cache: Settings > Privacy > Clear browsing data
- Disable extensions: Settings > Extensions
- Update: Settings > About Chrome

### Firefox
- Clear cache: Options > Privacy > Clear Data
- Disable add-ons: Add-ons Manager
- Update: Menu > Help > About Firefox

### Safari
- Clear cache: Preferences > Privacy > Manage Website Data
- Disable extensions: Preferences > Extensions
- Update: System Preferences > Software Update

### Edge
- Clear cache: Settings > Privacy > Choose what to clear
- Disable extensions: Extensions menu
- Update: Settings > About Microsoft Edge
