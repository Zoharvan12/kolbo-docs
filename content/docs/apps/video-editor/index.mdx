# Video Editor

Kolbo.AI's Video Editor is a complete, professional-grade video editing application that runs entirely in your browser, with deep integration into the Kolbo ecosystem.

## Overview

Edit videos like a pro without installing software. The Video Editor provides timeline-based editing, multi-track support, effects, transitions, and more - all in your browser.
