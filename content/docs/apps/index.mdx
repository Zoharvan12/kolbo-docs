# Apps

Kolbo.AI includes three powerful specialized applications designed for specific creative workflows.

## Overview

The Apps suite provides professional-grade applications:
- **AI Board**: Visual brainstorming and planning (similar to Miro)
- **AI Docs**: Document creation with AI and project context
- **Video Editor**: Complete browser-based video editing

Each app integrates deeply with Kolbo.AI's ecosystem, leveraging your projects, generated media, and AI capabilities.

## Available Apps

### AI Board
Visual collaboration and brainstorming:
- Infinite canvas workspace
- Sticky notes and shapes
- Mind mapping tools
- AI-powered ideation
- Team collaboration
- Template library

**Best for:**
- Brainstorming sessions
- Project planning
- User journey mapping
- Wireframing
- Visual organization
- Team workshops

### AI Docs
Intelligent document creation:
- Google Docs-like interface
- AI writing assistance
- Project context awareness
- Template system
- Collaboration features
- Export options

**Best for:**
- Content writing
- Project documentation
- Marketing copy
- Scripts and screenplays
- Reports and proposals
- Collaborative writing

### Video Editor
Professional video editing in browser:
- Timeline-based editing
- Multi-track support
- Effects and transitions
- Audio mixing
- Export options
- No software installation

**Best for:**
- Video post-production
- Marketing videos
- Social media content
- YouTube videos
- Podcasts with video
- Complete video projects
