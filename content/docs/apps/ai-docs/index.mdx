# AI Docs

AI Docs is Kolbo.AI's intelligent document editor that combines the familiarity of Google Docs with powerful AI assistance and deep integration with your Kolbo projects.

## Overview

Create, edit, and collaborate on documents with AI-powered writing assistance that understands your project context.

- Paragraph styles
- Lists and tables
- Images and media
- Headers and footers
