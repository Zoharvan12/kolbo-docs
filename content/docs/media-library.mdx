Centralized hub for organizing and managing all media created or uploaded in Kolbo.AI.

## What's in Media Library

All your media is automatically saved:
- Generated images (all image tools)
- Videos (all video tools)
- Audio files and music
- 3D models
- Uploaded files
- Chat artifacts
- Documents

## Organization

**Automatic Organization:**
- By media type (Images, Videos, Audio, Documents, 3D)
- By project
- By tool used
- By creation date

**Manual Organization:**
- Create folders
- Add custom tags
- Star favorites
- Create collections

## Search & Filter

**Search**: Text search across all media

**Filters:**
- Type (images, videos, audio, documents, 3D)
- Date range
- Project
- Tool used
- Tags
- Resolution, aspect ratio, file size

**Sort**: Date, name, size, type

## View Modes

**Grid View**: Thumbnail grid, visual browsing
**List View**: Detailed list with metadata
**Gallery View**: Full-screen slideshow

## Actions

**Single File:**
- Preview
- Download
- Edit in Canvas
- Add to project
- Share (generate link)
- Delete

**Bulk Operations:**
- Select multiple items
- Bulk download (ZIP)
- Bulk delete
- Bulk tag
- Bulk move to folder

## Integration

**Quick Actions:**
- Edit in Canvas
- Add to Video Editor
- Use in Training Lab
- Add to AI Board
- Create variations

## Use Cases

**Asset Management**: Organize by project/client, quick retrieval
**Client Work**: Create galleries, share selections, deliver finals
**Content Library**: Reusable assets, brand assets, templates
**Portfolio**: Curate best work, create galleries, share

## Best Practices

1. Tag consistently for powerful search
2. Organize by project for clarity
3. Use folders for logical structure
4. Bulk operations save time
5. Regular cleanup maintains efficiency
6. Star favorites for quick access
7. Delete unused media to free storage

## Storage Management

View storage usage by type and project. Delete unused files, archive old projects, or upgrade plan for more space.
