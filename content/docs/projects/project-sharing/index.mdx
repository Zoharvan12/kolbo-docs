# Project Sharing & Collaboration

Kolbo.AI's project sharing features enable seamless collaboration with team members, clients, and stakeholders.

## Overview

Share entire projects or specific content with others, control permissions, and collaborate in real-time on creative work.

## Sharing Options

### Project Sharing
Share entire projects:
- All project content
- All sessions
- Media library access
- Tool access
- Context sharing

### Content Sharing
Share specific items:
- Individual images/videos
- Specific sessions
- Gallery links
- Public sharing
- Limited access

### Folder Sharing
Share entire folders:
- Multiple projects
- Client folders
- Campaign folders
- Organizational sharing

## Sharing Workflows

### Share with Team
Collaborate internally:
1. Open project
2. Click "Share" button
3. Add team member email
4. Set permission level
5. Optional: Add message
6. Send invitation

### Share with Client
External sharing:
1. Select content to share
2. Create share link
3. Set view permissions
4. Optional: Password protect
5. Optional: Expiration date
6. Copy link to share

### Public Sharing
Share with anyone:
1. Select content
2. Enable public sharing
3. Generate public link
4. Optional: Embed code
5. Share link anywhere
