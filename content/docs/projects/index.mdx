# Projects & Organization

Kolbo.AI uses a powerful project-based organization system to keep all your creative work organized and accessible.

## Overview

Everything in Kolbo.AI is organized around **Projects**:
- Projects contain all related media and sessions
- Organize projects into folders
- Share projects with team members
- Access from anywhere
- Complete workspace management

## Key Concepts

### Projects
Main organizational unit:
- Container for all work
- Includes media from all tools
- Sessions within each tool
- Shared context across tools
- Project-specific settings

### Folders
Organize projects:
- Group related projects
- Nested folder structure
- Client folders
- Campaign folders
- Custom organization

### Sessions
Individual workspaces:
- Each tool creates sessions
- Multiple sessions per project
- Session history
- Resume where you left off

### Workspace
Your dashboard:
- Overview of all projects
- Recent activity
- Quick access
- Search across projects
