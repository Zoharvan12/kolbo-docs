# Image to Video

Transform static images into dynamic videos using Kolbo.AI's Image to Video tool with four unique animation modes.

## Overview

Image to Video brings your static images to life with AI-powered animation. Choose from four different modes depending on your needs:

- **Single Image**: Animate a single image
- **First and Last Frame**: Control start and end positions
- **Elements**: Animate multiple elements together
- **Draw to Video**: Draw motion paths for animation

## Animation Modes

### Single Image Mode
Upload one image and let AI animate it:
- Natural motion generation
- Camera movements
- Subject animation
- Environmental effects

**Best for:**
- Product shots
- Character poses
- Landscape photos
- Simple animations

### First and Last Frame Mode
Control animation precisely:
- Upload starting frame
- Upload ending frame
- AI generates smooth transition
- Precise control over motion

**Best for:**
- Controlled movements
- Specific transitions
- Predictable results
- Professional animations

### Elements Mode
Unique multi-element animation:
- Add multiple elements to one scene
- Animate each element independently
- Layer different animations
- Create complex compositions

**Best for:**
- Product showcases with multiple items
- Character interactions
- Complex scenes
- Creative storytelling

### Draw to Video Mode
Direct motion control:
- Draw the path of movement
- Specify animation direction
- Visual motion guidance
- Intuitive control

**Best for:**
- Specific motion paths
- Creative camera movements
- Directional animations
- Precise control needs
