# Text to Video

Create original video content from text descriptions using multiple AI models and powerful preset systems.

## Overview

Text to Video generates complete videos from your text descriptions. No images or videos required - just describe what you want to see.

## Generation Workflow

### 1. Write Your Prompt
Describe the video you want:
- Subject and action
- Environment and setting
- Camera angle and movement
- Style and mood
- Specific details

### 2. Choose Model & Presets
- Select AI model (or let Smart Agent choose)
- Apply presets for effects/movements
- Use Sora 2/Veo 3 for cinematic results

### 3. Set Parameters
- Duration: 2-10 seconds
- Aspect ratio: Various options
- Quality settings
- Number of variations

### 4. Generate & Refine
- Generate video
- Review results
- Regenerate with adjustments
- Use variations for different angles
