# Beginner Guides

Welcome to Kolbo.AI! These beginner guides will help you get started and master the basics.

## Getting Started

### Your First 5 Minutes
**Learn**: Platform basics
**Time**: 5 minutes
**Goal**: Understand Kolbo.AI

**Steps:**
1. Log in to your account
2. Take the welcome tour
3. Explore the interface
4. View example projects
5. Try the help widget

### Create Your First Image
**Learn**: Text to Image basics
**Time**: 10 minutes
**Goal**: Generate your first AI image

**Steps:**
1. Navigate to Image Tools > Text to Image
2. Write a simple prompt: "A cute cat sitting on a windowsill"
3. Click Generate
4. Wait for result
5. View in Media Library

**Tips:**
- Be descriptive
- Start simple
- Try different prompts
- Save your favorites

### Create Your First Project
**Learn**: Project management
**Time**: 10 minutes
**Goal**: Organize your work

**Steps:**
1. Click "New Project"
2. Name it "My First Project"
3. Generate some images
4. Browse your project
5. Organize your media

**What You'll Learn:**
- Creating projects
- Organizing work
- Finding your media
- Project navigation

### Writing Prompts
**Learn**: Effective prompt writing
**Time**: 20 minutes
**Goal**: Get better results

**Topics:**
- What makes a good prompt
- Descriptive language
- Style keywords
- Common patterns
- Iterating prompts

**Examples:**

**Basic Prompt:**
```
A dog
```

**Better Prompt:**
```
A golden retriever puppy playing in a sunny park, professional photography, natural lighting
```

**Advanced Prompt:**
```
A golden retriever puppy playing with a red ball in a sunny park, shallow depth of field, professional photography, golden hour lighting, vibrant colors, joyful mood
```

**Practice:**
1. Start with basic prompts
2. Add more details
3. Include style keywords
4. Try different variations
5. Compare results

## Basic Tools

### Text to Image Basics
**Learn**: Generate images from text
**Time**: 30 minutes
**Goal**: Master basic image generation

**Tutorial:**
1. Choose Text to Image tool
2. Write descriptive prompt
3. Select model (start with Flux)
4. Choose aspect ratio
5. Generate image
6. Try variations
7. Save to project

**Common Use Cases:**
- Social media graphics
- Blog illustrations
- Concept visualization
- Creative exploration
- Quick mockups

**Best Practices:**
- Be specific
- Include style keywords
- Try multiple models
- Generate variations
- Iterate based on results

## Basic Workflows

### Social Media Post Workflow
**Learn**: Create social media content
**Time**: 30 minutes
**Goal**: Complete social post

**Steps:**
1. Create new project
2. Generate image
3. Adjust aspect ratio for platform
4. Add text if needed (Canvas)
5. Export optimized
6. Ready to post!

**Platforms:**
- Instagram: 1:1 square or 4:5 portrait
- Facebook: 1:1 or 16:9
- Twitter: 16:9 or 1:1
- LinkedIn: 1:1 or 1.91:1

## Troubleshooting Basics

### Common Beginner Issues

**"I can't find my generated content"**
- Check Media Library
- Check project filter
- Check if generation completed
- Refresh page

**"Results don't match my prompt"**
- Add more details
- Be more specific
- Try different model
- Include style keywords
- Check examples

**"Generation failed"**
- Check credit balance
- Simplify prompt
- Try different model
- Check internet connection
- Contact support

**"Interface is slow"**
- Close unused tabs
- Clear browser cache
- Check internet speed
- Try different browser
