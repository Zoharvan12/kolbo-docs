# Media Library

The Media Library is your centralized hub for organizing, finding, and managing all media created or uploaded in Kolbo.AI.

## Overview

Every image, video, audio file, and document you create or upload in Kolbo.AI is automatically saved to your Media Library, providing a powerful system for organizing and retrieving your creative assets.

## Key Features

### Automatic Organization
Media is organized automatically:
- By media type (Images, Videos, Audio, Documents, 3D)
- By project
- By tool used
- By creation date
- By tags and labels
