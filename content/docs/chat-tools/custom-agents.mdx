Create specialized AI assistants with custom instructions and knowledge - similar to Custom GPTs.

## How to Create an Agent

1. Go to Chat Tools → "Create Agent"
2. Describe what you want the agent to do
3. Configure:
   - **Name**: Clear, descriptive (e.g., "Marketing Writer", "Code Reviewer")
   - **Instructions**: Define behavior, tone, and guidelines
   - **Icon**: Choose emoji or upload thumbnail
   - **Files**: Attach reference documents (PDF, DOC, CSV, JSON, code files)
4. Save and start using

## Features

- **Specialized behavior**: Consistent instructions every time
- **Custom knowledge**: Agents reference attached files
- **Shareable**: Share with team or import from others
- **Switchable**: Use different agents in same project

## Common Use Cases

**Marketing Writer**: Brand voice, product info, target audience
**Code Reviewer**: Coding standards, best practices, security rules
**Research Assistant**: Methodology, citation format, subject expertise
**Support Agent**: Product knowledge, policies, troubleshooting guides

## Best Practices

1. Be specific in instructions with examples
2. Attach focused, relevant files only
3. Test with various prompts and refine
4. Use clear naming conventions
5. Update files and instructions as needs change
