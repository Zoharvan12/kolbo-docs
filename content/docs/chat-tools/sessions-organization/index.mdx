# Sessions & Organization

Organize your chat conversations with sessions, folders, search, and sharing features.

## Chat Sessions

### What Are Sessions?

A session is an individual chat conversation:
- Contains message history
- Maintains context
- Can be renamed and organized
- Saved automatically
- Linked to a project

### Creating Sessions

**New Chat Session:**
1. Click "New Chat"
2. Start conversation
3. Session created automatically
4. Continue as long as needed

**Multiple Sessions:**
- Create multiple sessions per project
- Each session has independent context
- Organize by topic or purpose
- Switch between sessions easily

### Session Management

**Rename Sessions:**
1. Click on session
2. Select rename option
3. Give descriptive name
4. Makes finding easier

**Delete Sessions:**
1. Select session
2. Choose delete option
3. Confirm deletion
4. Session permanently removed

**Session List:**
- View all your sessions
- Organized by project
- Most recent first
- Quick access

## Session Organization

### Session Folders

Create folders to organize sessions:
- Group related chats
- Organize by topic, client, or project
- Nested folder support
- Drag and drop sessions

**Creating Folders:**
1. Go to sessions list
2. Click "New Folder"
3. Name your folder
4. Add sessions to folder

**Moving Sessions:**
1. Select session
2. Drag to folder
3. Or use move option
4. Organize as needed

### Naming Conventions

**Best Practices:**
- Use descriptive names
- Include date if relevant
- Mention topic or purpose
- Keep names concise

**Examples:**
- "Marketing Campaign Q1 2025"
- "Product Feature Ideas"
- "Client Meeting Notes"
- "Research: AI Trends"

## Session Sharing

### Share Chat Sessions

**How to Share:**
1. Open session
2. Click share button
3. Set permissions
4. Copy and send link

**View Only:**
- Read conversation
- See all messages
- No editing allowed

**Comment:**
- View conversation
- Add comments
- Discuss content
- No direct chat

**Edit/Collaborate:**
- View and add messages
- Continue conversation
- Full interaction
- Team collaboration

### Shared Sessions Management

## Troubleshooting

**Can't find session?**
- Use search feature
- Check folders
- Check different project
- Verify not deleted

**Sharing not working?**
- Check permissions
- Verify link copied correctly
- Check recipient access
- Contact support

**Sessions loading slowly?**
- Too many messages may slow load
- Consider starting new session
- Archive old sessions
- Contact support if persists

**Lost session?**
- Check trash/archive
- Search by content
- Check different project
- Contact support for recovery
