# Chat Tools

Kolbo.AI's Chat Tools provide a powerful, flexible chat interface with multi-model support, file analysis, custom agents, and intelligent memory features.

## Overview

Chat Tools offer advanced capabilities beyond traditional AI chat:
- **Multiple AI Models**: Switch between GPT, Claude, Gemini, and more
- **Custom Agents**: Create specialized AI assistants
- **File Analysis**: Upload and analyze videos, audio, images, documents
- **User Memories**: AI remembers your preferences and context
- **Artifacts**: View and edit generated code and websites
- **Sessions**: Organized, searchable conversations

## Key Features

### Multi-Model Chat

Chat with multiple AI models:
- **Single Mode**: One model at a time
- **Dual Mode**: Two models simultaneously
- **Triple Mode**: Three models at once

Access to multiple AI models from various leading providers.

**Switch models anytime** without losing conversation context.

### Custom Agents

Create specialized AI assistants with custom instructions, attach reference files, and share agents with your team. Like Custom GPTs, but integrated into your Kolbo workspace.

### File Upload & Analysis

Upload files directly to chat for AI analysis:
- **Video**: MP4, MOV, AVI, WebM (transcription & analysis)
- **Audio**: MP3, WAV, M4A (transcription)
- **Images**: JPG, PNG, WebP, GIF (vision & OCR)
- **Documents**: PDF, DOC, DOCX, TXT
- **Data**: CSV, JSON
- **Other**: SRT, and more

Plus **URL content fetching** and **YouTube video analysis**.

### User Memories

AI memory system that learns about you:
- Manually create or use AI-assisted generation
- Maximum 100 memories (optimal for AI context)
- Import from ChatGPT and other platforms
- Memories load at every conversation start

### Artifacts

View and edit AI-generated code and websites with a built-in code editor, live preview, and shareable links. Perfect for landing pages, email templates, and interactive content.
