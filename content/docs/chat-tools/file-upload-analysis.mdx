Upload files directly to chat for AI analysis, transcription, and understanding.

## Supported File Types

**Video**: MP4, MOV, AVI, WebM - transcribe, analyze, describe content
**Audio**: MP3, WAV, M4A - transcribe speech, summarize, extract information
**Images**: JPG, PNG, WebP, GIF - describe, OCR, identify objects
**Documents**: PDF, DOC/DOCX, TXT, SRT - read, extract, summarize
**Data**: CSV, JSON - analyze patterns, generate insights

## How to Upload

1. Click upload/attachment button in chat
2. Select files from your computer
3. Wait for upload to complete
4. Ask questions about the files

Upload multiple files at once - AI can reference all of them.

## Features

**Automatic Transcription**: Videos and audio files are automatically transcribed
**URL Fetching**: Paste any website or YouTube URL - AI fetches and analyzes content
**Multi-File Analysis**: Ask questions across multiple uploaded files
**OCR**: Read text from images and screenshots

## Common Uses

**For Videos/Audio:**
- "Transcribe this video"
- "What are the main points discussed?"
- "Summarize the conversation"

**For Documents:**
- "What are the key findings in this report?"
- "Extract the contact information"
- "Summarize the main arguments"

**For Images:**
- "What's in this image?"
- "Read the text from this screenshot"
- "Describe the layout"

**For Data:**
- "What trends do you see?"
- "What's the average of column X?"

**For URLs:**
- "Summarize this article: [URL]"
- "What does this YouTube video cover?"

## Best Practices

1. Use high-quality, clear files for best results
2. Clear audio = better transcription
3. Ask specific, targeted questions
4. Upload related files together
5. Wait for processing to complete (transcription takes time)

File size limits apply per plan. Very large files may take longer to process.
