# File Upload & Analysis

Upload files directly to chat for AI analysis, transcription, and understanding. Supports a wide variety of file types.

## Supported File Types

### Video Files
Upload video files for analysis and transcription:
- MP4, MOV, AVI, WebM
- Other common video formats

**What AI can do with videos:**
- Transcribe spoken content
- Analyze visual content
- Describe what's happening
- Extract information
- Answer questions about the video

### Audio Files
Upload audio for transcription and analysis:
- MP3, WAV, M4A
- Other audio formats

**What AI can do with audio:**
- Transcribe speech to text
- Identify speakers
- Extract key information
- Summarize content
- Answer questions about audio

### Image Files
Upload images for visual understanding:
- JPG, PNG, WebP, GIF
- Other common image formats

**What AI can do with images:**
- Describe what's in the image
- Read text in images (OCR)
- Identify objects and people
- Analyze compositions
- Answer questions about images

### Document Files
Upload documents for reading and analysis:
- **PDF**: Read and extract text
- **DOC/DOCX**: Microsoft Word documents
- **TXT**: Plain text files
- **SRT**: Subtitle files

**What AI can do with documents:**
- Read and understand content
- Extract key information
- Summarize documents
- Answer questions about content
- Reference specific sections

### Data Files
Upload structured data:
- **CSV**: Spreadsheet data
- **JSON**: Structured data files

**What AI can do with data:**
- Analyze data patterns
- Generate insights
- Answer questions about data
- Create visualizations descriptions
- Process structured information

### Other File Types
You can try uploading other file types:
- Some formats may work
- Not 100% guaranteed
- AI will attempt to process
- Check if it works for your use case

## How to Upload Files

### Basic Upload
1. Click the upload/attachment button in chat
2. Select files from your computer
3. Wait for upload to complete
4. Files appear in the chat
5. Start asking questions about the files

### Multiple Files
- Upload multiple files at once
- AI can reference all uploaded files
- Ask questions across multiple files
- Combine information from different sources

### File Management
- View uploaded files in chat
- Remove files you no longer need
- Upload new files anytime
- Files stay in conversation context

## Using Uploaded Files

### Ask Questions
Once files are uploaded, you can:
- Ask specific questions about content
- Request summaries
- Extract information
- Compare information across files
- Get analysis and insights

- Video/audio transcription takes time

- Progress indicators show status

### File Types
- Core file types are fully supported
- Other formats may work but not guaranteed
- Try different formats if one doesn't work
- Contact support for specific needs

### Accuracy
- Transcription accuracy depends on audio quality
- Clear speech = better transcription
- Background noise affects accuracy
- Multiple speakers may be challenging
