# Text to Image

Create stunning images from text descriptions using Kolbo.AI's collection of image generation models.

## Overview

Text to Image is designed for pure image generation from text prompts. It supports multiple AI models and allows reference images for style guidance (not for editing).

## Available Models

- **Seedance**: High-quality realistic and artistic images
- **Reve**: Creative and stylized generations
- **GPT Image**: OpenAI's image generation
- **Ideogram**: Text rendering and graphic design
- **Nano Banana**: Specialized model
- **Qwen**: Chinese-developed AI model
- **Grok Imagine**: X.ai's image model
- **Flux**: Fast, high-quality generation

## Key Features

### Text Prompting
- Natural language descriptions
- Style modifiers and keywords
- Negative prompts
- Prompt templates

### Reference Images
- Use images as style reference
- Not for editing (use Image Editing tool for that)
- Guide the AI's aesthetic direction
- Combine multiple reference styles

### Settings & Controls
- Aspect ratio selection
- Number of variations
- Quality settings
- Model-specific parameters

### Generation Options
- Single images
- Multiple variations
- Batch generation
- Quick iterations

## Best Practices

**Writing Effective Prompts:**
- Be specific and descriptive
- Include style keywords (realistic, artistic, cinematic, etc.)
- Describe lighting and mood
- Mention specific details you want

**Using Reference Images:**
- Use for style inspiration, not editing
- Combine multiple references for unique styles
- Experiment with different models
