# Canvas

Combine AI-powered editing with manual image editing tools in one workspace.

## AI Editing Tools

**Upscale:**
Increase image resolution and clarity

**Face Swap:**
Replace faces in images

**Reframe:**
Change aspect ratio intelligently

**Remove Background:**
Extract subjects from backgrounds

**Inpaint:**
Edit specific areas of images

**Enhance Skin:**
Improve portrait quality

## Manual Editing Tools

**Text:**
Add text overlays to images

**Images:**
Add other images as layers

**Draw:**
Freehand drawing tools

**Adjustments:**
Lightroom-style controls:
- Brightness
- Contrast
- Saturation
- Exposure
- Highlights/Shadows
- Temperature
- Tint
- And more

## Workflow

1. Upload or select image
2. Choose AI tool or manual tool
3. Make adjustments
4. Preview changes
5. Apply and download

## Sessions

Canvas uses sessions to organize your work:
- Each session contains one image with edit history
- Switch between sessions
- Save and return to edits later
- Organize by project

## Features

- Non-destructive editing
- Layer-based workflow
- Undo/redo support
- Real-time preview
- Export in various formats
