# Image Tools

Generate and edit images using AI with multiple specialized tools and models.

## Overview

Kolbo.AI provides six image generation and editing tools:
- **Text to Image**: Generate images from text prompts
- **Creative Director**: Generate complete creative sets in one command
- **Angels**: Multi-angle image generation from single prompt
- **Training Lab**: Train custom models on your own characters/styles
- **Canvas**: AI + manual image editing tools
- **Custom Models**: Pre-trained Kolbo models for specific styles

## Shared Features Across All Tools

### Moodboards

Moodboards work across **all image tools** in Kolbo.AI:
- Create collections of reference images
- Apply moodboard style to any generation (Text to Image, Training Lab, Canvas, etc.)
- Manage and organize moodboards by project
- Reuse moodboards across different tools
- Share moodboards with team members

**How it works:**
1. Create a moodboard with reference images
2. Select the moodboard in any image tool
3. The AI uses the moodboard's visual style to guide generation
4. Consistent style across all your image creations

### Image Presets

Image Presets are available across **all image generation tools**:
- Quick style templates for instant application
- Save time with pre-configured settings
- Apply consistent styles across tools
- Create custom presets for your workflows
- Share presets with team

**Benefits:**
- Use the same preset in Text to Image, Training Lab, and Canvas
- Maintain brand consistency across all tools
- Speed up your workflow with saved settings
- Experiment with different preset combinations
