# Image Tools

Generate and edit images using AI with multiple specialized tools and models.

## Overview

Kolbo.AI provides six image generation and editing tools:
- **Text to Image**: Generate images from text prompts
- **Creative Director**: Generate complete creative sets in one command
- **Angels**: Multi-angle image generation from single prompt
- **Training Lab**: Train custom models on your own characters/styles
- **Canvas**: AI + manual image editing tools
- **Custom Models**: Pre-trained Kolbo models for specific styles
