# Angels

Generate multiple angles of the same subject from a single prompt - a unique Kolbo.AI feature.

## What It Does

Angels generates the same subject from different camera angles:
- Front view
- Side view  
- Back view
- 3/4 views
- Top-down or bottom-up angles

All from one prompt, with consistent style and subject.

## How to Use

1. Describe your subject in detail
2. Specify which angles you want
3. Generate - all angles created together
4. All images maintain consistency

## Benefits

- **Consistency**: Same subject, character, or product across all angles
- **Complete Coverage**: Get all perspectives in one generation
- **Time-Saving**: No need to regenerate for each angle
- **Perfect for**: Product photography, character design, 3D reference

## Use Cases

**Product Photography:**
Show products from all sides for e-commerce

**Character Design:**
Create character turnarounds for animation or 3D modeling

**Architecture:**
Display buildings or interiors from multiple viewpoints

**3D Reference:**
Generate reference images for 3D modeling projects
