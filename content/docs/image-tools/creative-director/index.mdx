# Creative Director

Generate complete creative sets or variations for products/characters in a single command.

## What It Does

Creative Director generates multiple related images in one generation:
- Product variations (different angles, contexts, lighting)
- Character variations (different poses, expressions, outfits)
- Complete creative sets for campaigns
- Consistent style across all generated images

## How to Use

1. Describe your product or character
2. Specify what type of variations you want
3. Set quantity and preferences
4. Generate - all variations created together

## Benefits

- **Consistency**: All images share the same style and character
- **Speed**: Generate complete sets in one go
- **Variety**: Get multiple perspectives automatically
- **Efficiency**: No need to generate each variation separately

## Examples

**Product variations:**
"Coffee mug - show product from front, side, and in use"

**Character variations:**
"Fantasy warrior - create standing pose, action pose, and portrait"

**Campaign set:**
"Smartphone - lifestyle shots for social media campaign"
